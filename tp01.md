1. Sistema de recomendaciones de Netflix

Nombre del producto o servicio:
Netflix – Sistema de recomendaciones.

Qué problema resuelve con ML:
Netflix tiene una gran cantidad de películas y series. Si el usuario tuviera que buscar manualmente qué ver, sería muy difícil decidir. Por eso utilizan machine learning para recomendar contenido que probablemente le guste al usuario según lo que ya vio antes.

Tipo de ML que usa y por qué:
Probablemente utiliza aprendizaje supervisado, porque el sistema se entrena con datos donde ya se conoce el resultado. Por ejemplo, si un usuario vio una serie completa o le dio "me gusta", eso sirve como información para predecir qué otras series podrían gustarle.

Datos que probablemente necesita:

Historial de series y películas vistas

Géneros de contenido

Tiempo de visualización

Valoraciones o likes

Comportamiento de usuarios con gustos similares

2. Detección de fraude en PayPal

Nombre del producto o servicio:
PayPal – Sistema de detección de fraude.

Qué problema resuelve con ML:
Las plataformas de pago online reciben miles de transacciones por segundo y algunas pueden ser fraudulentas. El sistema analiza esas operaciones para detectar comportamientos sospechosos y evitar fraudes.

Tipo de ML que usa y por qué:
Probablemente utiliza una combinación de:

Aprendizaje supervisado, porque se entrena con transacciones ya clasificadas como fraude o legítimas.

Aprendizaje no supervisado, para detectar comportamientos extraños que no coinciden con patrones normales.

Datos que probablemente necesita:

Historial de transacciones

Ubicación desde donde se realiza el pago

Dispositivo utilizado

Monto de la transacción

Comportamiento normal del usuario

3. Recomendaciones de productos en Amazon

Nombre del producto o servicio:
Amazon – Sistema de recomendación de productos.

Qué problema resuelve con ML:
Amazon utiliza machine learning para recomendar productos a los usuarios mientras navegan por la tienda. Esto facilita encontrar productos que podrían interesar al cliente y mejora la experiencia de compra.

Tipo de ML que usa y por qué:
Principalmente aprendizaje supervisado, ya que el sistema aprende a partir de compras anteriores y del comportamiento de los usuarios para predecir qué productos podrían comprar.

Datos que probablemente necesita:

Historial de compras

Productos vistos

Búsquedas realizadas

Productos agregados al carrito

Compras de otros usuarios similares

4. Avancargo (empresa argentina de logística)

Nombre del producto o servicio:
Avancargo – Plataforma logística digital.

Qué problema resuelve con ML:
Avancargo es una empresa tecnológica argentina que conecta empresas que necesitan transportar cargas con transportistas disponibles. Utiliza análisis de datos y machine learning para optimizar rutas, tiempos de entrega y asignación de transportistas.

Tipo de ML que usa y por qué:
Probablemente aprendizaje supervisado, porque el sistema puede aprender de datos históricos de envíos para predecir cuál es la mejor ruta o cuánto tardará una entrega.

Datos que probablemente necesita:

Origen y destino de los envíos

Historial de tiempos de entrega

Información del tráfico o rutas

Tipo de carga

Disponibilidad de transportistas

5. Spotify – Descubrimiento de música

Nombre del producto o servicio:
Spotify – Recomendación musical (Discover Weekly, Daily Mix, etc.).

Qué problema resuelve con ML:
Spotify utiliza machine learning para recomendar música nueva a los usuarios según sus gustos. Esto ayuda a descubrir artistas o canciones que el usuario probablemente no encontraría por sí solo.

Tipo de ML que usa y por qué:
Probablemente utiliza aprendizaje no supervisado y supervisado.

No supervisado para encontrar patrones entre usuarios con gustos musicales similares.

Supervisado para predecir qué canciones podrían gustarle a un usuario en particular.

Datos que probablemente necesita:

Canciones escuchadas

Listas de reproducción

Artistas favoritos

Tiempo de escucha

Comportamiento de usuarios con gustos similares
